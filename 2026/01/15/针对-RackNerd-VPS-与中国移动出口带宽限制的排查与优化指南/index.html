й’ҲеҜ№ RackNerd VPS дёҺдёӯеӣҪз§»еҠЁеҮәеҸЈеёҰе®ҪйҷҗеҲ¶зҡ„жҺ’жҹҘдёҺеҲ©з”ЁHysteria 2дјҳеҢ–жҢҮеҚ—

1. й—®йўҳиғҢжҷҜдёҺзҺ°иұЎ

еңЁдҪҝз”Ё RackNerd зӯүзҫҺиҘҝе»үд»· VPS ж—¶пјҢдёӯеӣҪз§»еҠЁе®ҪеёҰз”ЁжҲ·еёёйҒҮеҲ°д»ҘдёӢе…ёеһӢзҺ°иұЎпјҡ

  • еҲқжңҹжӯЈеёёпјҡVPS еҲҡйғЁзҪІе®ҢжҲҗж—¶пјҲйҖҡеёёеңЁйқһй«ҳеі°ж—¶ж®өпјүпјҢиҝһжҺҘйҖҹеәҰеҝ«пјҢе»¶иҝҹеҸҜжҺҘеҸ—гҖӮ
  • еҗҺжңҹиЎ°еҮҸпјҡеңЁжҷҡй«ҳеі°пјҲ19:00 - 23:00пјүжҲ–йғЁзҪІдёҖж®өж—¶й—ҙеҗҺпјҢSSH иҝһжҺҘеҚЎйЎҝпјҢж–Үд»¶дј иҫ“йҖҹеәҰи·ҢиҮіжһҒдҪҺж°ҙе№іпјҲеҰӮ 0.5Mbps - 2MbpsпјүпјҢдё”дјҙйҡҸй«ҳдёўеҢ…зҺҮгҖӮ
  • еҚҸи®®еӨұж•ҲпјҡдҪҝз”ЁеҹәдәҺ TCP зҡ„еҚҸи®®пјҲеҰӮ VLESS-RealityгҖҒVMess-TCPпјүж—¶пјҢеҚідҪҝжӣҙжҚўз«ҜеҸЈжҲ–дјӘиЈ…еҹҹеҗҚпјҢйҖҹеәҰдҫқз„¶ж— жі•жҒўеӨҚгҖӮ

2. жҺ’жҹҘжҖқи·ҜдёҺж №еӣ еҲҶжһҗ

еҪ“еҮәзҺ°дёҠиҝ°й—®йўҳж—¶пјҢйңҖжҢүз…§д»ҘдёӢйҖ»иҫ‘иҝӣиЎҢжҺ’жҹҘпјҢзЎ®и®ӨжҳҜеҗҰдёәзәҝи·ҜжӢҘе өеҜјиҮҙзҡ„зү©зҗҶз“¶йўҲгҖӮ

2.1 жҺ’йҷӨжң¬ең°дёҺжңҚеҠЎз«Ҝй…ҚзҪ®й—®йўҳ

еңЁиҝӣиЎҢй“ҫи·ҜеҲҶжһҗеүҚпјҢйңҖзЎ®и®Өе№¶йқһеҚ•зӮ№ж•…йҡңпјҡ

  • жң¬ең°зҪ‘з»ңпјҡи®ҝй—®еӣҪеҶ…зҪ‘з«ҷжҲ–е…¶д»–йқһи·ЁеўғжңҚеҠЎйҖҹеәҰжӯЈеёёпјҢжҺ’йҷӨжң¬ең°е…үзҢ«жҲ–и·Ҝз”ұеҷЁй—®йўҳгҖӮ
  • VPS иҙҹиҪҪпјҡйҖҡиҝҮ top жҲ– htop е‘Ҫд»ӨжЈҖжҹҘ VPS зҡ„ CPU е’ҢеҶ…еӯҳеҚ з”ЁгҖӮеҰӮжһң CPU Steal Time (st) иҫғдҪҺпјҢжҺ’йҷӨвҖңйӮ»еұ…вҖқеҚ з”Ёиө„жәҗеҜјиҮҙзҡ„жҖ§иғҪдёӢйҷҚгҖӮ

2.2 й“ҫи·Ҝи·Ҝз”ұиҝҪиёӘ (TraceRoute)

иҝҷжҳҜе®ҡдҪҚй—®йўҳзҡ„ж ёеҝғжӯҘйӘӨгҖӮйҖҡиҝҮ tracert (Windows) жҲ– mtr (Linux/macOS) е·Ҙе…·иҝҪиёӘж•°жҚ®еҢ…и·Ҝеҫ„гҖӮ

е…ёеһӢж•…йҡңж ·жң¬еҲҶжһҗпјҲеҹәдәҺдёӯеӣҪз§»еҠЁпјүпјҡ

  1. зңҒеҶ…йӘЁе№ІзҪ‘пјҲз¬¬ 1-7 и·іпјүпјҡе»¶иҝҹиҫғдҪҺпјҲ<30msпјүпјҢж— дёўеҢ…гҖӮиҜҙжҳҺжң¬ең°еҲ°зңҒзә§еҮәеҸЈзҪ‘з»ңйҖҡз•…гҖӮ
  2. еӣҪйҷ…еҮәеҸЈиҠӮзӮ№пјҲз¬¬ 8-9 и·іпјүпјҡ
    • IP ж®өйҖҡеёёдёә 223.120.*.* жҲ– 221.183.*.*гҖӮ
    • зҺ°иұЎпјҡе»¶иҝҹеҮәзҺ°зӘҒеўһпјҲдҫӢеҰӮд»Һ 30ms йЈҷеҚҮиҮі 200ms+пјүпјҢдё”дјҙйҡҸиҜ·жұӮи¶…ж—¶пјҲRequest Timed OutпјүгҖӮ
    • з»“и®әпјҡиҝҷжҳҜдёӯеӣҪз§»еҠЁзҡ„еӣҪйҷ…еҮәеҸЈзҪ‘е…ігҖӮжӯӨеӨ„зҡ„й«ҳе»¶иҝҹе’ҢдёўеҢ…ж„Ҹе‘ізқҖеҮәеҸЈеёҰе®ҪжӢҘе өгҖӮ
  3. еўғеӨ–жүҝиҪҪзҪ‘пјҲз¬¬ 10+ и·іпјүпјҡ
    • иҝӣе…Ҙ cogentco.com жҲ–зұ»дјје»үд»·йӘЁе№ІзҪ‘иҝҗиҗҘе•ҶгҖӮ
    • зҺ°иұЎпјҡжҢҒз»ӯзҡ„дёўеҢ…пјҲжҳҹеҸ· *пјүе’Ңй«ҳе»¶иҝҹгҖӮ
    • з»“и®әпјҡRackNerd зӯүе»үд»· VPS е•Ҷе®¶йҖҡеёёжҺҘе…Ҙ Cogent зӯүжҲҗжң¬иҫғдҪҺзҡ„зәҝи·ҜпјҢиҝҷдәӣзәҝи·ҜдёҺдёӯеӣҪиҝҗиҗҘе•Ҷзҡ„дә’иҒ”дә’йҖҡиҙЁйҮҸеңЁй«ҳеі°жңҹжһҒе·®гҖӮ

2.3 дёәд»Җд№Ҳ TCP еҚҸи®®пјҲReality/VMessпјүдјҡеӨұж•Ҳпјҹ

Reality зӯүеҚҸи®®еә•еұӮеҹәдәҺ TCPгҖӮTCP еҚҸи®®жӢҘжңүжӢҘеЎһжҺ§еҲ¶жңәеҲ¶пјҲCongestion Controlпјүпјҡ

  1. еҪ“й“ҫи·ҜеҸ‘з”ҹдёўеҢ…ж—¶пјҢTCP еҚҸи®®е°Ҷе…¶еҲӨе®ҡдёәзҪ‘з»ңжӢҘе өгҖӮ
  2. еҸ‘йҖҒз«Ҝдјҡдё»еҠЁйҷҚдҪҺеҸ‘йҖҒзӘ—еҸЈеӨ§е°ҸпјҲеҮҸйҖҹпјүд»Ҙзј“и§ЈжӢҘе өгҖӮ
  3. еңЁз§»еҠЁеҮәеҸЈжӢҘе өеҜјиҮҙзҡ„й«ҳдёўеҢ…зҺҜеўғдёӢпјҲдёўеҢ…зҺҮеҸҜиғҪй«ҳиҫҫ 20%-75%пјүпјҢTCP дјҡдёҚж–ӯи§ҰеҸ‘еҮҸйҖҹжңәеҲ¶пјҢеҜјиҮҙе®һйҷ…еҗһеҗҗйҮҸжһҒдҪҺпјҲеҰӮеҚЎеңЁ Kbps зә§еҲ«пјүпјҢз”ҡиҮіж–ӯиҝһгҖӮ

3. и§ЈеҶіж–№жЎҲпјҡйғЁзҪІ Hysteria 2

еңЁж— жі•ж”№еҸҳзү©зҗҶзәҝи·ҜпјҲдёҚжӣҙжҚў CN2 GIA зӯүй«ҳз«Ҝзәҝи·Ҝпјүзҡ„еүҚжҸҗдёӢпјҢжӣҙжҚўдј иҫ“еұӮеҚҸи®®жҳҜе”ҜдёҖжңүж•Ҳзҡ„жҠҖжңҜжүӢж®өгҖӮ

3.1 дёәд»Җд№ҲйҖүжӢ© Hysteria 2

Hysteria 2 еҹәдәҺ UDP еҚҸи®®гҖӮдёҺ TCP дёҚеҗҢпјҢUDP жң¬иә«дёҚдҝқиҜҒеҸҜйқ дј иҫ“пјҢд№ҹдёҚе…·еӨҮејәеҲ¶зҡ„йҖҖйҒҝжңәеҲ¶гҖӮHysteria 2 йҖҡиҝҮиҮӘе®ҡд№үзҡ„жӢҘеЎһжҺ§еҲ¶з®—жі•пјҢеңЁжЈҖжөӢеҲ°дёўеҢ…ж—¶пјҢдёҚдјҡеғҸ TCP йӮЈж ·еӨ§е№…йҷҚдҪҺеҸ‘йҖҒйҖҹзҺҮпјҢиҖҢжҳҜеҲ©з”ЁеҶ—дҪҷеҸ‘еҢ…жҲ–еҝ«йҖҹйҮҚдј жҠўеҚ еёҰе®ҪгҖӮ

  • йҖӮз”ЁеңәжҷҜпјҡй«ҳдёўеҢ…гҖҒй«ҳе»¶иҝҹгҖҒзәҝи·ҜжӢҘе өзҡ„вҖңеһғеңҫвҖқзәҝи·ҜгҖӮ
  • ж•ҲжһңпјҡеңЁ TCP д»…иғҪи·‘ 2Mbps зҡ„зҺҜеўғдёӢпјҢHysteria 2 еҸҜиғҪи·‘еҮә 50Mbps-100Mbps зҡ„йҖҹеәҰгҖӮ

3.2 йғЁзҪІеүҚеҮҶеӨҮ

  • еҹҹеҗҚпјҡе»әи®®жӢҘжңүдёҖдёӘеҹҹеҗҚи§ЈжһҗеҲ° VPS IPпјҲз”ЁдәҺз”іиҜ· SSL иҜҒд№ҰпјҢжҸҗй«ҳеҚҸи®®йҡҗи”ҪжҖ§пјүгҖӮ
  • йҳІзҒ«еўҷпјҡHysteria 2 дҫқиө– UDP з«ҜеҸЈпјҢйңҖеңЁ VPS еҗҺеҸ°еҸҠзі»з»ҹеҶ…йғЁж”ҫиЎҢ TCP е’Ң UDP з«ҜеҸЈгҖӮ

ж”ҫиЎҢз«ҜеҸЈе‘Ҫд»Ө (д»Ҙ CentOS/AlmaLinux зҡ„ firewalld дёәдҫӢ)пјҡ

3.3 е®ҳж–№и„ҡжң¬йғЁзҪІжӯҘйӘӨ

дҪҝз”Ё Hysteria е®ҳж–№жҸҗдҫӣзҡ„дёҖй”®и„ҡжң¬иҝӣиЎҢе®үиЈ…гҖӮ

1. жү§иЎҢе®үиЈ…е‘Ҫд»Ө

  • еҰӮжһңи„ҡжң¬иҜўй—®жҳҜеҗҰе®үиЈ…пјҢйҖүжӢ© YesгҖӮ
  • е®үиЈ…е®ҢжҲҗеҗҺпјҢи„ҡжң¬дјҡиҮӘеҠЁеҲӣе»әзі»з»ҹжңҚеҠЎ hysteria-server.serviceгҖӮ

2. й…ҚзҪ®ж–Үд»¶зј–еҶҷ
е®ҳж–№и„ҡжң¬е®үиЈ…еҗҺпјҢйңҖжүӢеҠЁй…ҚзҪ® /etc/hysteria/config.yamlгҖӮдёәйҒҝе…Қж јејҸй”ҷиҜҜпјҢе»әи®®дҪҝз”Ё cat е‘Ҫд»ӨзӣҙжҺҘеҶҷе…ҘгҖӮ

жіЁж„ҸпјҡеҰӮжһңдёҚдҪҝз”Ё Cloudflare CDNпјҢиҜ·зЎ®дҝқеҹҹеҗҚеңЁ DNS и§Јжһҗж—¶е…ій—ӯд»ЈзҗҶпјҲеҚі Cloudflare зҡ„вҖңе°Ҹй»„дә‘вҖқйңҖдёәзҒ°иүІпјүпјҢд»…еҒҡ DNS и§ЈжһҗгҖӮ

й…ҚзҪ®жЁЎжқҝпјҲACME иҮӘеҠЁиҜҒд№ҰзүҲпјүпјҡ
е°Ҷд»ҘдёӢеҶ…е®№дёӯзҡ„ еҹҹеҗҚгҖҒйӮ®з®ұ е’Ң еҜҶз Ғ жӣҝжҚўдёәе®һйҷ…еҖјгҖӮ

3. еӨ„зҗҶз«ҜеҸЈеҚ з”Ё
RackNerd зҡ„зі»з»ҹжЁЎжқҝеҸҜиғҪйў„иЈ…дәҶ httpd жҲ– nginx еҚ з”Ё 80 з«ҜеҸЈпјҢеҜјиҮҙ ACME иҜҒд№Ұз”іиҜ·еӨұиҙҘгҖӮйңҖжё…зҗҶз«ҜеҸЈпјҡ

4. еҗҜеҠЁжңҚеҠЎдёҺйӘҢиҜҒ

  • зҠ¶жҖҒжЈҖжҹҘпјҡиӢҘзҠ¶жҖҒдёә Active: active (running)пјҢдё”ж—Ҙеҝ—жҳҫзӨә ACME иҜҒд№ҰиҺ·еҸ–жҲҗеҠҹпјҢеҚійғЁзҪІе®ҢжҲҗгҖӮ
  • й”ҷиҜҜжҺ’жҹҘпјҡиӢҘеҗҜеҠЁеӨұиҙҘпјҢдҪҝз”Ё journalctl -u hysteria-server.service -n 20 жҹҘзңӢж—Ҙеҝ—гҖӮеёёи§ҒеҺҹеӣ дёә YAML зј©иҝӣй”ҷиҜҜжҲ– 80/443 з«ҜеҸЈд»Қиў«еҚ з”ЁгҖӮ

4. е®ўжҲ·з«Ҝй…ҚзҪ®иҰҒзӮ№

жңҚеҠЎз«ҜйғЁзҪІжҲҗеҠҹеҗҺпјҢе®ўжҲ·з«ҜпјҲеҰӮ v2rayNгҖҒNekoBoxпјүзҡ„й…ҚзҪ®иҮіе…ійҮҚиҰҒгҖӮ

  1. еҹәзЎҖдҝЎжҒҜпјҡ
    • ең°еқҖпјҡдҪ зҡ„еҹҹеҗҚ
    • з«ҜеҸЈпјҡ443
    • еҚҸи®®пјҡHysteria 2
    • SNIпјҡдҪ зҡ„еҹҹеҗҚ
    • еҜҶз Ғпјҡй…ҚзҪ®ж–Үд»¶дёӯи®ҫзҪ®зҡ„еҜҶз Ғ
  2. дёҚе®үе…ЁиҝһжҺҘ (Allow Insecure)пјҡ
    • иӢҘдҪҝз”Ё ACME з”іиҜ·зҡ„жӯЈејҸиҜҒд№ҰпјҢжӯӨйЎ№е…ій—ӯпјҲи®ҫзҪ®дёә FalseпјүгҖӮ
    • иӢҘдҪҝз”ЁиҮӘзӯҫиҜҒд№ҰпјҢжӯӨйЎ№ејҖеҗҜпјҲи®ҫзҪ®дёә TrueпјүгҖӮ
  3. еёҰе®Ҫи®ҫзҪ® (е…ій”®)пјҡ
    • еңЁе®ўжҲ·з«Ҝй…ҚзҪ®дёӯпјҢеҝ…йЎ»еЎ«еҶҷдёӢиЎҢеёҰе®Ҫ/дёӢиҪҪйҖҹеәҰпјҲеҰӮ 100 Mbps жҲ– 500 MbpsпјүгҖӮ
