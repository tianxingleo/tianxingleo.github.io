з”Ё GLOMAP жӣҝжҚў COLMAP еҠ йҖҹ 3DGS йҮҚе»әз®Ўзәҝ

еңЁ 3D Gaussian Splatting (3DGS) зҡ„ж•°жҚ®йў„еӨ„зҗҶйҳ¶ж®өпјҢзЁҖз–ҸйҮҚе»әпјҲSparse ReconstructionпјүеҫҖеҫҖжҳҜиҖ—ж—¶з“¶йўҲгҖӮдј з»ҹзҡ„ COLMAP йҮҮз”ЁеўһйҮҸејҸпјҲIncrementalпјүйҮҚе»әзӯ–з•ҘпјҢиҷҪз„¶зІҫеәҰй«ҳдҪҶйҡҸзқҖеӣҫзүҮж•°йҮҸеўһеҠ йҖҹеәҰе‘ҲжҢҮж•°зә§дёӢйҷҚгҖӮ

GLOMAP дҪңдёәдёҖз§Қе…Ёж–°зҡ„е…ЁеұҖејҸпјҲGlobalпјүSfM зі»з»ҹпјҢиғҪеӨҹеңЁдҝқжҢҒйҮҚе»әиҙЁйҮҸзҡ„еҗҢж—¶пјҢе°ҶйҮҚе»әйҖҹеәҰжҸҗеҚҮ 1-2 дёӘж•°йҮҸзә§пјҢдё”еҜ№ејұзә№зҗҶе’Ңйҡҫж ·жң¬е…·жңүжӣҙејәзҡ„йІҒжЈ’жҖ§гҖӮжң¬ж–Үе°Ҷд»Ӣз»ҚеҰӮдҪ•еңЁзҺ°д»Ј Linux ејҖеҸ‘зҺҜеўғдёӯпјҢд»ҘжңҖз®ҖжҙҒзҡ„ж–№ејҸзј–иҜ‘ GLOMAP е№¶е°Ҷе…¶ж— зјқйӣҶжҲҗеҲ° Python иҮӘеҠЁеҢ–з®ЎзәҝдёӯгҖӮ

1. ж ёеҝғжҖқи·Ҝ

GLOMAP зҡ„и®ҫи®ЎзҗҶеҝөжҳҜе…је®№ COLMAP зҡ„ж•°жҚ®ж јејҸгҖӮеӣ жӯӨпјҢжҲ‘д»¬дёҚйңҖиҰҒжӣҝжҚўж•ҙдёӘз®ЎзәҝпјҢеҸӘйңҖжӣҝжҚўеҗҺз«Ҝзҡ„ж ёеҝғйҮҚе»әжЁЎеқ—пјҡ

  1. еүҚз«ҜпјҲдҝқжҢҒдёҚеҸҳпјүпјҡз»§з»ӯдҪҝз”Ё COLMAP иҝӣиЎҢзү№еҫҒжҸҗеҸ–пјҲFeature Extractorпјүе’ҢеҢ№й…ҚпјҲMatcherпјүгҖӮ
  2. еҗҺз«ҜпјҲжӣҝжҚўпјүпјҡе°Ҷ colmap mapper жӣҝжҚўдёә glomap mapperгҖӮ
  3. иҫ“е…Ҙ/иҫ“еҮәпјҡGLOMAP зӣҙжҺҘиҜ»еҸ– COLMAP зҡ„ database.dbпјҢе№¶иҫ“еҮәж ҮеҮҶзҡ„ COLMAP зЁҖз–ҸжЁЎеһӢж јејҸпјҲ.binпјүгҖӮ

2. зҺҜеўғдҫқиө–дёҺзј–иҜ‘ (The вҖңGoldenвҖқ Configuration)

еңЁж··еҗҲдәҶ Conda зҺҜеўғпјҲж–°зүҲ ABIпјүе’Ңзі»з»ҹзҺҜеўғпјҲж—§зүҲ ABI/GCC 14пјүзҡ„еӨҚжқӮеңәжҷҜдёӢпјҢзј–иҜ‘ GLOMAP жһҒжҳ“йҒҮеҲ° ABI еҶІзӘҒжҲ–дҫқиө–ең°зӢұгҖӮ

з»ҸиҝҮйӘҢиҜҒпјҢвҖңзі»з»ҹзә§дҫқиө– + Conda зҺҜеўғвҖқ зҡ„ж··еҗҲзј–иҜ‘зӯ–з•ҘжҳҜжңҖзЁіеҒҘзҡ„ж–№жЎҲгҖӮ

2.1 е®үиЈ…зі»з»ҹзә§дҫқиө–

дёҚиҰҒиҜ•еӣҫеңЁ Conda еҶ…йғЁи§ЈеҶіеӣҫеҪўеӯҰдҫқиө–пјҢзӣҙжҺҘдҪҝз”Ё apt е®үиЈ…ж ёеҝғеә“пјҢзЎ®дҝқ OpenGL е’Ң ABI е…је®№жҖ§пјҡ

2.2 зј–иҜ‘ GLOMAP

еңЁ Conda зҺҜеўғжҝҖжҙ»зҡ„зҠ¶жҖҒдёӢпјҢдҪҝз”Ёд»ҘдёӢ CMake й…ҚзҪ®гҖӮиҝҷдёӘй…ҚзҪ®ејәеҲ¶ GLOMAP й“ҫжҺҘзі»з»ҹзҡ„ Boost е’Ң OpenGL еә“пјҢеҗҢж—¶е…је®№ GCC 14 зҡ„ж–°зү№жҖ§пјҢйҒҝе…ҚдәҶ undefined reference е’ҢеӨҙж–Үд»¶еҶІзӘҒгҖӮ

  • жіЁпјҡе®үиЈ…е®ҢжҲҗеҗҺпјҢйӘҢиҜҒ glomap --help жҳҜеҗҰиғҪжӯЈеёёиҫ“еҮәгҖӮ

3. Python з®ЎзәҝйӣҶжҲҗ

еңЁиҮӘеҠЁеҢ–и„ҡжң¬пјҲеҰӮ run_glomap.pyпјүдёӯпјҢжҲ‘д»¬йңҖиҰҒеҒҡдёӨд»¶дәӢпјҡжіЁе…ҘиҝҗиЎҢж—¶зҺҜеўғеҸҳйҮҸд»Ҙи§ЈеҶіеә“еҶІзӘҒпјҢе№¶дҝ®ж”№и°ғз”ЁйҖ»иҫ‘гҖӮ

3.1 иҝҗиЎҢж—¶зҺҜеўғжіЁе…Ҙ (и§ЈеҶі DLL Hell)

GLOMAP е’Ң COLMAP еҸҜиғҪдҫқиө–дёҚеҗҢзүҲжң¬зҡ„ libsqlite3 жҲ– libstdc++гҖӮдёәдәҶйҳІжӯўиҝҗиЎҢж—¶еҙ©жәғпјҲеҰӮ SQL logic errorпјүпјҢеҝ…йЎ»ејәеҲ¶дјҳе…ҲеҠ иҪҪ Conda зҺҜеўғзҡ„еҠЁжҖҒеә“гҖӮ

еңЁ Python и„ҡжң¬зҡ„е…ҘеҸЈеӨ„ж·»еҠ пјҡ

3.2 жӣҝжҚў Mapper и°ғз”ЁйҖ»иҫ‘

жүҫеҲ°еҺҹжң¬и°ғз”Ё colmap mapper зҡ„д»Јз Ғж®өпјҢе°Ҷе…¶жӣҝжҚўдёә glomap mapperгҖӮ

жіЁж„Ҹи·Ҝеҫ„е·®ејӮпјҡCOLMAP йңҖиҰҒжҢҮе®ҡеҲ° sparse/0пјҢиҖҢ GLOMAP еҸӘйңҖжҢҮе®ҡеҲ° sparseпјҲе®ғдјҡиҮӘеҠЁеҲӣе»ә 0 еӯҗзӣ®еҪ•пјүгҖӮ

4. жҖ»з»“

йҖҡиҝҮдёҠиҝ°ж”№йҖ пјҢжҲ‘д»¬е®һзҺ°дәҶдёҖдёӘй«ҳжҖ§иғҪзҡ„ 3DGS йў„еӨ„зҗҶз®Ўзәҝпјҡ

  1. зј–иҜ‘еұӮйқўпјҡйҖҡиҝҮж··еҗҲй“ҫжҺҘзӯ–з•ҘпјҲSystem Boost + Conda DependenciesпјүпјҢи§ЈеҶідәҶ GCC 14 е’Ң CUDA зҺҜеўғдёӢзҡ„ ABI е…је®№жҖ§й—®йўҳгҖӮ
  2. иҝҗиЎҢеұӮйқўпјҡйҖҡиҝҮ LD_LIBRARY_PATH жіЁе…ҘпјҢи§ЈеҶідәҶ SQLite зӯүеә•еұӮеә“зҡ„зүҲжң¬еҶІзӘҒгҖӮ
  3. йҖ»иҫ‘еұӮйқўпјҡеҲ©з”Ё GLOMAP зҡ„е…ЁеұҖдјҳеҢ–з®—жі•жӣҝд»ЈдәҶдј з»ҹзҡ„еўһйҮҸејҸйҮҚе»әгҖӮ

иҝҷз§Қж–№жЎҲдёҚд»…еӨ§е№…зј©зҹӯдәҶеӨ§и§„жЁЎеңәжҷҜзҡ„йҮҚе»әж—¶й—ҙпјҢиҝҳжҳҫи‘—жҸҗеҚҮдәҶиҮӘеҠЁеҢ–жөҒзЁӢзҡ„зЁіе®ҡжҖ§гҖӮ

иё©еқ‘е®һеҪ•пјҡеңЁ GCC 14 дёҺ Conda ж··еҗҲзҺҜеўғдёӯзј–иҜ‘ GLOMAP зҡ„иҮіжҡ—ж—¶еҲ»

иғҢжҷҜпјҡ зӣ®ж ҮжҳҜе°Ҷдј з»ҹзҡ„ COLMAP еўһйҮҸејҸйҮҚе»әжӣҝжҚўдёә GLOMAP е…ЁеұҖйҮҚе»әд»ҘеҠ йҖҹ 3DGS з®ЎзәҝгҖӮ зҺҜеўғпјҡ

  • OS: Ubuntu 24.04 (WSL2)
  • Compiler: GCC 14.3.0 (Conda иҮӘеёҰпјҢжһҒж–°)
  • CUDA: 12.8
  • Base Environment: Conda (vggt)

рҹӣ‘ з¬¬дёҖе…іпјҡжһ„е»әзі»з»ҹзҡ„зүҲжң¬д»ЈжІҹ

1.1 CMake зүҲжң¬зҡ„вҖңиғҢеҲәвҖқ

иө·еҲқпјҢдёәдәҶиҝҪжұӮжңҖж–°зү№жҖ§пјҢжҲ‘иҝҗиЎҢдәҶ pip install cmake --upgradeпјҢе®үиЈ…дәҶ CMake 4.2.0пјҲйў„и§ҲзүҲпјүгҖӮ

жҠҘй”ҷзҺ°иұЎпјҡ

CMake

ж·ұеәҰеҲҶжһҗпјҡ GLOMAP еңЁзј–иҜ‘ж—¶дјҡиҮӘеҠЁдёӢиҪҪ COLMAP жәҗз ҒдҪңдёәдҫқиө–гҖӮCOLMAP зҡ„ CMake и„ҡжң¬иҝҳеңЁдҪҝз”Ёж—§зүҲзҡ„ FindBoost жЁЎеқ—жҹҘжүҫ Boost еә“гҖӮз„¶иҖҢпјҢCMake 4.x зүҲжң¬жҝҖиҝӣең°з§»йҷӨдәҶиҝҷдәӣеәҹејғжЁЎеқ—пјҢеҜјиҮҙеҗ‘еҗҺе…је®№жҖ§ж–ӯиЈӮгҖӮ

рҹ”§ дҝ®жӯЈпјҡ йҷҚзә§еӣһзЁіеҒҘзҡ„ 3.x зүҲжң¬зі»еҲ—гҖӮ

Bash

рҹӣ‘ з¬¬дәҢе…іпјҡдҫқиө–еә“зҡ„вҖңйҡҗиә«вҖқдёҺеҶІзӘҒ (OpenGL & CGAL)

2.1 Conda зј–иҜ‘еҷЁзҡ„вҖңеӮІж…ўвҖқ (OpenGL зјәеӨұ)

иҷҪз„¶зі»з»ҹйҖҡиҝҮ apt install libgl1-mesa-dev е®үиЈ…дәҶ OpenGLпјҢдҪҶ CMake дҫқз„¶жҠҘй”ҷпјҡ Could NOT find OpenGL (missing: OPENGL_opengl_LIBRARY OPENGL_glx_LIBRARY)

ж·ұеәҰеҲҶжһҗпјҡ Conda зҺҜеўғеҶ…зҡ„ CMake й»ҳи®Өжҗңзҙўи·Ҝеҫ„иў«йҡ”зҰ»еңЁ Conda зҺҜеўғеҶ…пјҲ$CONDA_PREFIXпјүгҖӮе®ғвҖңзңӢдёҚи§ҒвҖқе®ҝдё»жңә /usr/lib дёӢзҡ„зі»з»ҹеә“гҖӮ

вқҢ й”ҷиҜҜе°қиҜ•пјҡ иҜ•еӣҫж·»еҠ  -DCMAKE_PREFIX_PATH="/usr"гҖӮ еҗҺжһңпјҡCMake зЎ®е®һжүҫеҲ°дәҶ OpenGLпјҢдҪҶе®ғеҗҢж—¶д№ҹйЎәзқҖ /usr жүҫеҲ°дәҶзі»з»ҹиҮӘеёҰзҡ„ж—§зүҲ CUDA (12.0)пјҢеҜјиҮҙдёҺ Conda еҶ…зҡ„ж–°зүҲ CUDA (12.8) еҶІзӘҒпјҢжҠҘй”ҷ Found unsuitable version "12.0.140", but required is at least "12.6.85"гҖӮ

рҹ”§ дҝ®жӯЈпјҡ зІҫеҮҶжҠ•е–ӮгҖӮдёҚејҖж”ҫж•ҙдёӘ /usrпјҢиҖҢжҳҜжҳҫејҸжҢҮе®ҡеә“ж–Үд»¶и·Ҝеҫ„пјҡ

Bash

2.2 CGAL зҡ„жӯ»еҫӘзҺҜ

COLMAP ејәдҫқиө– CGALгҖӮе°қиҜ•еңЁ Conda еҶ…е®үиЈ… conda install cgalпјҢеҚҙеӣ  Python зүҲжң¬е’Ңдҫқиө–й”Ғжӯ»еҜјиҮҙ LibMambaUnsatisfiableErrorгҖӮдәҺжҳҜиҪ¬иҖҢдҪҝз”Ёзі»з»ҹзә§ CGAL (apt install libcgal-dev)гҖӮ

жҠҘй”ҷзҺ°иұЎпјҡ зј–иҜ‘ж—¶жҠҘй”ҷ fatal error: CGAL/Delaunay_triangulation_3.h: No such file or directoryгҖӮ

ж·ұеәҰеҲҶжһҗпјҡ CMake иҷҪз„¶еңЁй…ҚзҪ®йҳ¶ж®өжүҫеҲ°дәҶеә“пјҲLinker еұӮйқўпјүпјҢдҪҶ Conda иҮӘеёҰзҡ„зј–иҜ‘еҷЁпјҲCompiler еұӮйқўпјүдёәдәҶйҳІжӯўзҺҜеўғжұЎжҹ“пјҢеұҸи”ҪдәҶ /usr/include ж ҮеҮҶеӨҙж–Үд»¶и·Ҝеҫ„гҖӮ

рҹ”§ дҝ®жӯЈ (иҪҜй“ҫжҺҘеӨ§жі•)пјҡ ж¬әйӘ—зј–иҜ‘еҷЁпјҢе°Ҷзі»з»ҹеӨҙж–Үд»¶жҳ е°„еҲ° Conda зҺҜеўғдёӯпјҡ

Bash

рҹӣ‘ з¬¬дёүе…іпјҡзј–иҜ‘еҷЁзҡ„вҖңи¶…еүҚвҖқ (GCC 14 vs Boost)

иҝҷжҳҜжңҖйҡҗи”Ҫзҡ„дёҖдёӘеқ‘гҖӮ

жҠҘй”ҷзҺ°иұЎпјҡ

C++

ж·ұеәҰеҲҶжһҗпјҡ

  • GCC 14пјҡеҺҹз”ҹж”ҜжҢҒе№¶е®ҡд№үдәҶ 128 дҪҚжө®зӮ№ж•° (__float128) зҡ„ numeric_limits зү№еҢ–гҖӮ
  • Boost 1.82пјҡдёҚзҹҘйҒ“ GCC 14 е·Із»Ҹе№ІдәҶиҝҷдәӢпјҢдәҺжҳҜе®ғиҮӘе·ұеҸҲе®ҡд№үдәҶдёҖйҒҚгҖӮ
  • з»“жһңпјҡC++ д№ҹе°ұжҳҜжүҖи°“зҡ„ One Definition Rule (ODR) иҝқи§„пјҢзј–иҜ‘зӣҙжҺҘзӮёиЈӮгҖӮ

вқҢ й”ҷиҜҜе°қиҜ•пјҡ дҪҝз”Ёзј–иҜ‘еҸӮж•° -DBOOST_MATH_DISABLE_FLOAT128гҖӮж— ж•ҲпјҢеӣ дёәжҹҗдәӣ Boost еӨҙж–Үд»¶зҡ„еҶ…йғЁеҢ…еҗ«йҖ»иҫ‘еҸҜиғҪз»•иҝҮдәҶиҝҷдёӘе®ҸжЈҖжҹҘгҖӮ

рҹ”§ дҝ®жӯЈ (жәҗз Ғ Patch)пјҡ зј–еҶҷ Python и„ҡжң¬пјҢзӣҙжҺҘдҝ®ж”№ Conda зҺҜеўғйҮҢзҡ„ Boost еӨҙж–Үд»¶гҖӮ
