微分方程速成笔记

PART I 概念与基础

  1. 定义：含有自变量 xxx、未知函数 yyy 及其各阶导数的方程。
  2. 阶：方程中导数的最高阶数。
  3. 解的分类：
    • 通解：含有独立任意常数的个数与方程阶数相同的解（例如一阶方程含1个 CCC，二阶含2个 C1,C2C_1, C_2C1​,C2​）。
    • 特解：不含任意常数或任意常数确定后的解。
    • 初始条件：用于确定通解中任意常数的条件（定解条件）。
  4. 线性微分方程：yyy 和它的各阶导数都是一次的，且系数都是 xxx 的函数。
    • 例：y′′−2y=3y'' - 2y = 3y′′−2y=3 （线性）
    • 例：y′=y2+1y' = y^2 + 1y′=y2+1 （非线性，因为 yyy 是二次）
    • 例：y′=sin⁡y+6yy' = \sin y + 6yy′=siny+6y （非线性，因为 sin⁡y\sin ysiny）
    • 例：y′y=1y' y = 1y′y=1 （非线性，相乘算二次）

PART II 一阶微分方程

1. 可分离变量的微分方程

识别：y′=f(x)g(y)y' = f(x)g(y)y′=f(x)g(y) 或能化为 P(x)dx+Q(y)dy=0P(x)dx + Q(y)dy = 0P(x)dx+Q(y)dy=0 的形式，即 xxx 和 yyy 能彻底分开。

解法：

  1. 将 y′y'y′ 写为 dydx\frac{dy}{dx}dxdy​。
  2. 分离变量：dyg(y)=f(x)dx\frac{dy}{g(y)} = f(x)dxg(y)dy​=f(x)dx。
  3. 两边同时积分：∫dyg(y)=∫f(x)dx\int \frac{dy}{g(y)} = \int f(x)dx∫g(y)dy​=∫f(x)dx。

注意：求解通解时，默认分母不为0，无需讨论；若求全部解，需考虑分母为0的情况（可能会丢奇解）。

【例1】 求 1−x2y′=1−y2\sqrt{1-x^2}y' = \sqrt{1-y^2}1−x2​y′=1−y2​ 的通解。

解：

  1. 变形：1−x2dydx=1−y2\sqrt{1-x^2}\frac{dy}{dx} = \sqrt{1-y^2}1−x2​dxdy​=1−y2​
  2. 分离：dy1−y2=dx1−x2\frac{dy}{\sqrt{1-y^2}} = \frac{dx}{\sqrt{1-x^2}}1−y2​dy​=1−x2​dx​
  3. 积分：两边积分得 arcsin⁡y=arcsin⁡x+C\arcsin y = \arcsin x + Carcsiny=arcsinx+C

注：y=±1y=\pm 1y=±1 为奇解（包络），通解中不包含，但在求通解题目中一般不写。

2. 齐次方程

识别：y′=g(yx)y' = g(\frac{y}{x})y′=g(xy​)，或者方程中各项关于 x,yx, yx,y 的次数相同（例如都是三次项）。

识别：y′+p(x)y=q(x)y' + p(x)y = q(x)y′+p(x)y=q(x)。

  • y′y'y′ 和 yyy 是一次，系数仅含 xxx。

5. 伯努利方程

  3. 解出 zzz 后回代。

PART III 二阶可降阶微分方程

类型1：缺 y 型

识别：y′′=f(x,y′)y'' = f(x, y')y′′=f(x,y′) （方程中不显含 yyy）。

解法：令 y′=p(x)y' = p(x)y′=p(x)，则 y′′=p′y'' = p'y′′=p′。降阶为关于 ppp 的一阶方程。

【例7】 xy′′+3y′=0xy'' + 3y' = 0xy′′+3y′=0 通解。

解：

  1. 令 y′=py'=py′=p，则 xp′+3p=0xp' + 3p = 0xp′+3p=0。
  2. 分离变量：dpp=−3xdx⇒ln⁡∣p∣=−3ln⁡∣x∣+C1⇒p=Cx−3\frac{dp}{p} = -\frac{3}{x}dx \Rightarrow \ln|p| = -3\ln|x| + C_1 \Rightarrow p = Cx^{-3}pdp​=−x3​dx⇒ln∣p∣=−3ln∣x∣+C1​⇒p=Cx−3。
  3. 再积分：y′=Cx−3⇒y=∫Cx−3dx=−C2x−2+C2y' = Cx^{-3} \Rightarrow y = \int Cx^{-3}dx = -\frac{C}{2}x^{-2} + C_2y′=Cx−3⇒y=∫Cx−3dx=−2C​x−2+C2​。
  4. 整理常数：y=C1x2+C2y = \frac{C_1}{x^2} + C_2y=x2C1​​+C2​。

识别：y′′=f(y,y′)y'' = f(y, y')y′′=f(y,y′) （方程中不显含 xxx）。

【例11】 求 y′′+4y′+4y=e−2xy'' + 4y' + 4y = e^{-2x}y′′+4y′+4y=e−2x 的通解。

解：

  1. 求齐次通解：λ2+4λ+4=0⇒λ1,2=−2\lambda^2 + 4\lambda + 4 = 0 \Rightarrow \lambda_{1,2} = -2λ2+4λ+4=0⇒λ1,2​=−2。Y=(C1+C2x)e−2xY = (C_1 + C_2 x)e^{-2x}Y=(C1​+C2​x)e−2x。
  2. 求特解：右边是 e−2xe^{-2x}e−2x，即 α=−2\alpha = -2α=−2。
    因 −2-2−2 是二重特征根，故 k=2k=2k=2。
    设 y∗=Ax2e−2xy^* = A x^2 e^{-2x}y∗=Ax2e−2x。
    求导代入原方程（计算略），解得 A=1/2A = 1/2A=1/2。y∗=12x2e−2xy^* = \frac{1}{2}x^2 e^{-2x}y∗=21​x2e−2x。
  3. 总通解：y=(C1+C2x)e−2x+12x2e−2xy = (C_1 + C_2 x)e^{-2x} + \frac{1}{2}x^2 e^{-2x}y=(C1​+C2​x)e−2x+21​x2e−2x。

【例12】 求 y′′−2y′+5y=−12excos⁡2xy'' - 2y' + 5y = -\frac{1}{2}e^x \cos 2xy′′−2y′+5y=−21​excos2x 的通解。

解：

  1. 求齐次通解：λ2−2λ+5=0⇒λ=1±2i\lambda^2 - 2\lambda + 5 = 0 \Rightarrow \lambda = 1 \pm 2iλ2−2λ+5=0⇒λ=1±2i。α=1,β=2\alpha=1, \beta=2α=1,β=2。 Y=ex(C1cos⁡2x+C2sin⁡2x)Y = e^x(C_1 \cos 2x + C_2 \sin 2x)Y=ex(C1​cos2x+C2​sin2x)。
  2. 求特解：右边对应 α=1,β=2\alpha=1, \beta=2α=1,β=2，即 α±βi=1±2i\alpha \pm \beta i = 1 \pm 2iα±βi=1±2i 正好是特征根，故 k=1k=1k=1。
    设 y∗=xex(Acos⁡2x+Bsin⁡2x)y^* = x e^x (A \cos 2x + B \sin 2x)y∗=xex(Acos2x+Bsin2x)。
    (注意：虽然右边只有cos，特解里sin和cos都要设)
    代入原方程比较系数解得 A=0,B=1/8A=0, B=1/8A=0,B=1/8（此为示例结果，具体需计算）。y∗=18xexsin⁡2xy^* = \frac{1}{8} x e^x \sin 2xy∗=81​xexsin2x。
  3. 总通解：y=ex(C1cos⁡2x+C2sin⁡2x)+18xexsin⁡2xy = e^x(C_1 \cos 2x + C_2 \sin 2x) + \frac{1}{8} x e^x \sin 2xy=ex(C1​cos2x+C2​sin2x)+81​xexsin2x。

若方程右边是两项之和 f(x)=f1(x)+f2(x)f(x) = f_1(x) + f_2(x)f(x)=f1​(x)+f2​(x)，则特解 y∗=y1∗+y2∗y^* = y_1^* + y_2^*y∗=y1∗​+y2∗​，分别针对 f1f_1f1​ 和 f2f_2f2​ 求特解再相加。

PART V 高阶与欧拉方程（详细版）

这一部分主要处理两类问题：一是阶数 n≥3n \ge 3n≥3 的常系数线性方程，二是虽然是变系数但有特定结构的欧拉方程。

一、 高阶常系数线性齐次微分方程

1. 定义与核心思路

  • 形式：y(n)+an−1y(n−1)+⋯+a1y′+a0y=0y^{(n)} + a_{n-1}y^{(n-1)} + \dots + a_1 y' + a_0 y = 0y(n)+an−1​y(n−1)+⋯+a1​y′+a0​y=0（aia_iai​ 为常数）。
  • 解法：与二阶完全一致，核心是解 nnn 次特征方程：λn+an−1λn−1+⋯+a1λ+a0=0 \lambda^n + a_{n-1}\lambda^{n-1} + \dots + a_1 \lambda + a_0 = 0 λn+an−1​λn−1+⋯+a1​λ+a0​=0 虽然高次方程难解，但考试通常会给出可以因式分解的形式。

2. 特征根与通解的对应关系表

【例13】 求 y(4)−2y′′′+5y′′=0y^{(4)} - 2y''' + 5y'' = 0y(4)−2y′′′+5y′′=0 的通解。

【解题过程】

第一步：写出特征方程
将导数阶数换成 λ\lambdaλ 的次数：

λ4−2λ3+5λ2=0 \lambda^4 - 2\lambda^3 + 5\lambda^2 = 0 λ4−2λ3+5λ2=0

第二步：因式分解求特征根
提取公因式 λ2\lambda^2λ2：

λ2(λ2−2λ+5)=0 \lambda^2 (\lambda^2 - 2\lambda + 5) = 0 λ2(λ2−2λ+5)=0

由此得到两组根：

  1. λ2=0⇒λ1=λ2=0\lambda^2 = 0 \Rightarrow \lambda_1 = \lambda_2 = 0λ2=0⇒λ1​=λ2​=0。这是 **二重实根 0**。
  2. λ2−2λ+5=0\lambda^2 - 2\lambda + 5 = 0λ2−2λ+5=0。由求根公式或配方 (λ−1)2=−4(\lambda-1)^2 = -4(λ−1)2=−4，解得 λ3,4=1±2i\lambda_{3,4} = 1 \pm 2iλ3,4​=1±2i。这是 **一对共轭复根**（α=1,β=2\alpha=1, \beta=2α=1,β=2）。

第三步：拼写通解

  • 对应二重根 000 的部分：(C1+C2x)e0x=C1+C2x(C_1 + C_2 x)e^{0x} = C_1 + C_2 x(C1​+C2​x)e0x=C1​+C2​x。
  • 对应复根 1±2i1 \pm 2i1±2i 的部分：e1⋅x(C3cos⁡2x+C4sin⁡2x)e^{1\cdot x}(C_3 \cos 2x + C_4 \sin 2x)e1⋅x(C3​cos2x+C4​sin2x)。

最终结果：

y=C1+C2x+ex(C3cos⁡2x+C4sin⁡2x) y = C_1 + C_2 x + e^x(C_3 \cos 2x + C_4 \sin 2x) y=C1​+C2​x+ex(C3​cos2x+C4​sin2x)

二、 欧拉方程

这是考研及高数竞赛中唯一需要掌握的一类高阶变系数线性微分方程。

1. 识别题型

  • 特征：每一项中，xxx 的次数与 导数的阶数 严格一致。
  • 标准形式：xny(n)+⋯+a2x2y′′+a1xy′+a0y=f(x) x^n y^{(n)} + \dots + a_2 x^2 y'' + a_1 x y' + a_0 y = f(x) xny(n)+⋯+a2​x2y′′+a1​xy′+a0​y=f(x)

2. 解法：算子法（Operator Method）
相比于死记硬背代换公式，使用微分算子 DDD 更容易推导和记忆。

  • 换元：令 x=etx = e^tx=et （默认 x>0x>0x>0），则 t=ln⁡xt = \ln xt=lnx。
    我们将方程从关于 xxx 的变系数方程，转化为关于 ttt 的常系数方程。
  • 引入算子：记 D=ddtD = \frac{d}{dt}D=dtd​ （注意是对 ttt 求导）。
  • 代换规则（口诀：阶数是多少，就乘到 DDD 减几）：
    • yyy 保持不变 →y\to y→y
    • xy′x y'xy′ 换成 →Dy\to Dy→Dy
    • x2y′′x^2 y''x2y′′ 换成 →D(D−1)y\to D(D-1)y→D(D−1)y
    • x3y′′′x^3 y'''x3y′′′ 换成 →D(D−1)(D−2)y\to D(D-1)(D-2)y→D(D−1)(D−2)y
    • 以此类推…

3. 固定步骤

  1. 令 x=etx = e^tx=et。
  2. 将原方程中的 xky(k)x^k y^{(k)}xky(k) 全部替换为 DDD 算子形式。
  3. 化简得到关于 y(t)y(t)y(t) 的常系数线性微分方程。
  4. 求解出 y=f(t)y = f(t)y=f(t)。
  5. 回代：将 ttt 换回 ln⁡x\ln xlnx，将 ete^tet 换回 xxx。

【例14】 求欧拉方程 x2d2ydx2+4xdydx+2y=0 (x>0)x^2 \frac{d^2y}{dx^2} + 4x \frac{dy}{dx} + 2y = 0 \ (x>0)x2dx2d2y​+4xdxdy​+2y=0 (x>0) 的通解。

【解题过程】

第一步：换元与代入算子
令 x=etx = e^tx=et (t=ln⁡xt = \ln xt=lnx)，记 D=ddtD = \frac{d}{dt}D=dtd​。
原方程逐项替换：

  • x2y′′→D(D−1)yx^2 y'' \to D(D-1)yx2y′′→D(D−1)y
  • 4xy′→4Dy4x y' \to 4Dy4xy′→4Dy
  • 2y→2y2y \to 2y2y→2y

代入方程得：

D(D−1)y+4Dy+2y=0 D(D-1)y + 4Dy + 2y = 0 D(D−1)y+4Dy+2y=0

第二步：整理为常系数方程
提取 yyy，整理算子多项式：

(D2−D+4D+2)y=0 (D^2 - D + 4D + 2)y = 0 (D2−D+4D+2)y=0 (D2+3D+2)y=0 (D^2 + 3D + 2)y = 0 (D2+3D+2)y=0

这实际上就是关于 ttt 的二阶常系数齐次方程：

d2ydt2+3dydt+2y=0 \frac{d^2y}{dt^2} + 3\frac{dy}{dt} + 2y = 0 dt2d2y​+3dtdy​+2y=0

第三步：求解常系数方程
写出特征方程：

λ2+3λ+2=0 \lambda^2 + 3\lambda + 2 = 0 λ2+3λ+2=0

因式分解：(λ+1)(λ+2)=0(\lambda + 1)(\lambda + 2) = 0(λ+1)(λ+2)=0
特征根：λ1=−1,λ2=−2\lambda_1 = -1, \lambda_2 = -2λ1​=−1,λ2​=−2

写出关于 ttt 的通解：

y=C1e−t+C2e−2t y = C_1 e^{-t} + C_2 e^{-2t} y=C1​e−t+C2​e−2t

第四步：回代整理
因为 x=etx = e^tx=et，所以：

  • e−t=1et=1xe^{-t} = \frac{1}{e^t} = \frac{1}{x}e−t=et1​=x1​
  • e−2t=1(et)2=1x2e^{-2t} = \frac{1}{(e^t)^2} = \frac{1}{x^2}e−2t=(et)21​=x21​

代入通解表达式：

y=C1⋅1x+C2⋅1x2 y = C_1 \cdot \frac{1}{x} + C_2 \cdot \frac{1}{x^2} y=C1​⋅x1​+C2​⋅x21​

最终结果：

y=C1x+C2x2 y = \frac{C_1}{x} + \frac{C_2}{x^2} y=xC1​​+x2C2​​
