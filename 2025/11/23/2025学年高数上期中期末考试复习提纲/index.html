3. 应用：用泰勒公式求高阶导数

(对应图片 122206.png)

核心思想： 利用泰勒展开式的唯一性，对比系数求导数。

泰勒展开的通项系数为 f(n)(0)n!\frac{f^{(n)}(0)}{n!}n!f(n)(0)​，若能写出 xnx^nxn 的系数 ana_nan​，则 f(n)(0)=an⋅n!f^{(n)}(0) = a_n \cdot n!f(n)(0)=an​⋅n!。

  1. 展开 ln⁡(1+x)\ln(1+x)ln(1+x)：

    ln⁡(1+x)=x−12x2+⋯+(−1)n−3n−2xn−2+(−1)n−2n−1xn−1+…\ln(1+x) = x - \frac{1}{2}x^2 + \dots + \frac{(-1)^{n-3}}{n-2}x^{n-2} + \frac{(-1)^{n-2}}{n-1}x^{n-1} + \dotsln(1+x)=x−21​x2+⋯+n−2(−1)n−3​xn−2+n−1(−1)n−2​xn−1+…

  2. 代入 f(x)f(x)f(x) 并乘进 x2x^2x2：

    f(x)=x2[⋯+(−1)n−3n−2xn−2+… ]=⋯+(−1)n−3n−2xn+…f(x) = x^2 \left[ \dots + \frac{(-1)^{n-3}}{n-2}x^{n-2} + \dots \right] = \dots + \frac{(-1)^{n-3}}{n-2}x^n + \dotsf(x)=x2[⋯+n−2(−1)n−3​xn−2+…]=⋯+n−2(−1)n−3​xn+…

  3. 由泰勒展开式唯一性，对应 xnx^nxn 的系数：

    f(n)(0)n!=(−1)n−3n−2=(−1)n−1n−2\frac{f^{(n)}(0)}{n!} = \frac{(-1)^{n-3}}{n-2} = \frac{(-1)^{n-1}}{n-2}n!f(n)(0)​=n−2(−1)n−3​=n−2(−1)n−1​

2. 证明题综合应用实例

三角函数

1. 倒数与商数关系（基础铺垫）

这些在化简被积函数时最常用，比如把 tan⁡\tantan 拆成 sin⁡/cos⁡\sin/\cossin/cos。

  • tan⁡x=sin⁡xcos⁡x\tan x = \frac{\sin x}{\cos x}tanx=cosxsinx​
  • cot⁡x=cos⁡xsin⁡x=1tan⁡x\cot x = \frac{\cos x}{\sin x} = \frac{1}{\tan x}cotx=sinxcosx​=tanx1​
  • sec⁡x=1cos⁡x\sec x = \frac{1}{\cos x}secx=cosx1​
  • csc⁡x=1sin⁡x\csc x = \frac{1}{\sin x}cscx=sinx1​

2. 平方关系（勾股性质）

解不定积分时，常用于凑微分或三角代换（如 x=atan⁡tx=a\tan tx=atant）。

  1. sin⁡2x+cos⁡2x=1\sin^2 x + \cos^2 x = 1sin2x+cos2x=1 （万恶之源，最常用）
  2. 1+tan⁡2x=sec⁡2x1 + \tan^2 x = \sec^2 x1+tan2x=sec2x （重点：求 ∫sec⁡2xdx\int \sec^2 x dx∫sec2xdx 或 ∫tan⁡2xdx\int \tan^2 x dx∫tan2xdx 时必用）
  3. 1+cot⁡2x=csc⁡2x1 + \cot^2 x = \csc^2 x1+cot2x=csc2x

3. 倍角公式（简化计算的核心）

在求极限和化简导数时非常有用。

  • 正弦倍角：

    sin⁡2x=2sin⁡xcos⁡x\sin 2x = 2\sin x \cos xsin2x=2sinxcosx

  • 余弦倍角（有三种形式，灵活选择）：

    cos⁡2x=cos⁡2x−sin⁡2x\cos 2x = \cos^2 x - \sin^2 xcos2x=cos2x−sin2x cos⁡2x=2cos⁡2x−1\cos 2x = 2\cos^2 x - 1cos2x=2cos2x−1 cos⁡2x=1−2sin⁡2x\cos 2x = 1 - 2\sin^2 xcos2x=1−2sin2x

    tan⁡2x=2tan⁡x1−tan⁡2x\tan 2x = \frac{2\tan x}{1 - \tan^2 x}tan2x=1−tan2x2tanx​

4. 降幂公式（积分的救命稻草） ★★★

这是高数中最重要的一组变形！遇到 sin⁡2x\sin^2 xsin2x 或 cos⁡2x\cos^2 xcos2x 的积分，必须降幂才能积出来。

  • 正弦降幂：

    sin⁡2x=1−cos⁡2x2\sin^2 x = \frac{1 - \cos 2x}{2}sin2x=21−cos2x​

  • 余弦降幂：

    cos⁡2x=1+cos⁡2x2\cos^2 x = \frac{1 + \cos 2x}{2}cos2x=21+cos2x​

记忆口诀： “正弦减，余弦加”（看分子中间的符号）。

5. 和差角公式（展开项）

导数应用与积分基础

第一部分：概念辨析——什么“点”是坐标？什么“点”是数值？

(对应图片 203101.png, 203108.png)

这是做填空题最容易丢分的地方，务必区分清楚回答格式。

1. “非点”类：其实只是一个数 (xxx值)

虽然名字带“点”，但在数学定义上指的是定义域上的位置（横坐标）。回答时只需写 xxx 等于多少。

  • 驻点 (Stationary Point)
    • 定义：f′(x)=0f'(x) = 0f′(x)=0 的 xxx 值。
    • 回答格式： x=x0x = x_0x=x0​
  • 极值点 (Extremum Point)
    • 定义：函数取得极大值或极小值所对应的 xxx 值。
    • 回答格式： x=x0x = x_0x=x0​
    • 辨析：问“极值”答 yyy 值；问“极值点”答 xxx 值。
  • 间断点 (Discontinuity Point)
    • 定义：函数不连续的地方（如分母为0的点）。
    • 回答格式： x=x0x = x_0x=x0​
  • 不可导点
    • 回答格式： x=x0x = x_0x=x0​

2. “为点”类：必须写成坐标形式 (x,y)(x, y)(x,y)

指的是曲线（图像）上的一个几何点，必须由横纵坐标共同确定。

  • 拐点 (Inflection Point)
    • 定义：曲线凹凸性发生改变的点。
    • 回答格式： (x0,f(x0))(x_0, f(x_0))(x0​,f(x0​))
    • 严重警告： 很多同学求出了 x0x_0x0​ 直接填 x=x0x=x_0x=x0​，这在严格评分中通常是不得分的。必须代回原函数求出 y0y_0y0​，写成坐标对。

第二部分：拐点的判定与推广

(对应图片 203154.png)

1. 基础判定法

  • 第一充分条件： f(x)f(x)f(x) 在 x0x_0x0​ 处连续，且 f′′(x)f''(x)f′′(x) 在 x0x_0x0​ 两侧反号（一侧正一侧负）。
  • 第二充分条件： f′′(x0)=0f''(x_0) = 0f′′(x0​)=0 且 f′′′(x0)≠0f'''(x_0) \neq 0f′′′(x0​)=0。

2. 高阶导数推广判定（奇偶法则）

设在 x0x_0x0​ 处 nnn 阶可导，且满足：

f′(x0)=f′′(x0)=⋯=f(n−1)(x0)=0,f(n)(x0)≠0f'(x_0) = f''(x_0) = \dots = f^{(n-1)}(x_0) = 0, \quad f^{(n)}(x_0) \neq 0f′(x0​)=f′′(x0​)=⋯=f(n−1)(x0​)=0,f(n)(x0​)=0

即前 n−1n-1n−1 阶导数全为 0，第 nnn 阶不为 0：

  • 若 nnn 为 奇数 (3, 5, …) →\rightarrow→ (x0,f(x0))(x_0, f(x_0))(x0​,f(x0​)) 是 拐点。
  • 若 nnn 为 偶数 (2, 4, …) →\rightarrow→ x=x0x = x_0x=x0​ 是 极值点。
    • (f(n)>0f^{(n)} > 0f(n)>0 极小值，f(n)<0f^{(n)} < 0f(n)<0 极大值)

第三部分：曲率与曲率圆

(对应图片 203324.png)

1. 曲率计算公式 (KKK)

  • 直角坐标形式 (y=f(x)y=f(x)y=f(x))：

    K=∣y′′∣(1+y′2)3/2K = \frac{|y''|}{(1 + y'^2)^{3/2}}K=(1+y′2)3/2∣y′′∣​

  • 参数方程形式 ({x=x(t)y=y(t)\begin{cases} x=x(t) \\ y=y(t) \end{cases}{x=x(t)y=y(t)​):

    K=∣x′(t)y′′(t)−x′′(t)y′(t)∣(x′2(t)+y′2(t))3/2K = \frac{|x'(t)y''(t) - x''(t)y'(t)|}{(x'^2(t) + y'^2(t))^{3/2}}K=(x′2(t)+y′2(t))3/2∣x′(t)y′′(t)−x′′(t)y′(t)∣​

ρ=1K\rho = \frac{1}{K}ρ=K1​

  • 曲率中心坐标：

    α=x−y′(1+y′2)y′′,β=y+1+y′2y′′\alpha = x - \frac{y'(1+y'^2)}{y''}, \quad \beta = y + \frac{1+y'^2}{y''}α=x−y′′y′(1+y′2)​,β=y+y′′1+y′2​

    (x−α)2+(y−β)2=ρ2(x - \alpha)^2 + (y - \beta)^2 = \rho^2(x−α)2+(y−β)2=ρ2

第四部分：积分“滚瓜烂熟”表

注意：不定积分千万别忘了 +C+C+C ！

1. 基础函数积分

  1. ∫k dx=kx+C\int k \, dx = kx + C∫kdx=kx+C
  2. ∫xa dx=xa+1a+1+C(a≠−1)\int x^a \, dx = \frac{x^{a+1}}{a+1} + C \quad (a \neq -1)∫xadx=a+1xa+1​+C(a=−1)
  3. ∫1x dx=ln⁡∣x∣+C\int \frac{1}{x} \, dx = \ln|x| + C∫x1​dx=ln∣x∣+C **(注意绝对值)**
  4. ∫11+x2 dx=arctan⁡x+C\int \frac{1}{1+x^2} \, dx = \arctan x + C∫1+x21​dx=arctanx+C
  5. ∫11−x2 dx=arcsin⁡x+C\int \frac{1}{\sqrt{1-x^2}} \, dx = \arcsin x + C∫1−x2​1​dx=arcsinx+C
  6. ∫ax dx=axln⁡a+C\int a^x \, dx = \frac{a^x}{\ln a} + C∫axdx=lnaax​+C

2. 三角函数积分（核心组）

  1. ∫cos⁡x dx=sin⁡x+C\int \cos x \, dx = \sin x + C∫cosxdx=sinx+C
  2. ∫sin⁡x dx=−cos⁡x+C\int \sin x \, dx = -\cos x + C∫sinxdx=−cosx+C
  3. ∫sec⁡2x dx=tan⁡x+C\int \sec^2 x \, dx = \tan x + C∫sec2xdx=tanx+C **(常用)**
  4. ∫csc⁡2x dx=−cot⁡x+C\int \csc^2 x \, dx = -\cot x + C∫csc2xdx=−cotx+C
  5. ∫sec⁡xtan⁡x dx=sec⁡x+C\int \sec x \tan x \, dx = \sec x + C∫secxtanxdx=secx+C
  6. ∫csc⁡xcot⁡x dx=−csc⁡x+C\int \csc x \cot x \, dx = -\csc x + C∫cscxcotxdx=−cscx+C

  1. ∫sec⁡x dx=ln⁡∣sec⁡x+tan⁡x∣+C\int \sec x \, dx = \ln|\sec x + \tan x| + C∫secxdx=ln∣secx+tanx∣+C
  2. ∫csc⁡x dx=ln⁡∣csc⁡x−cot⁡x∣+C\int \csc x \, dx = \ln|\csc x - \cot x| + C∫cscxdx=ln∣cscx−cotx∣+C

第五部分：考前必背积分公式（进阶组）

(对应图片 204305.png)

1. 三角函数补充（对数型）

这两个公式通过换元法（凑微分）推导而来，建议直接背诵结论。

  1. 正切函数：

    ∫tan⁡x dx=−ln⁡∣cos⁡x∣+C\int \tan x \, dx = -\ln|\cos x| + C∫tanxdx=−ln∣cosx∣+C

    (记忆：tan⁡=sin⁡/cos⁡\tan = \sin/\costan=sin/cos，cos⁡\coscos 在分母求导是 −sin⁡-\sin−sin，所以有负号)

  2. 余切函数：

    ∫cot⁡x dx=ln⁡∣sin⁡x∣+C\int \cot x \, dx = \ln|\sin x| + C∫cotxdx=ln∣sinx∣+C

    (记忆：cot⁡=cos⁡/sin⁡\cot = \cos/\sincot=cos/sin，sin⁡\sinsin 在分母求导是 cos⁡\coscos，所以是正号)

2. 含参数 aaa 的重要积分（易混淆点）

这组公式是基础反三角函数积分的推广形式 (a>0)(a>0)(a>0)。

⚠️ 重点注意： 哪些公式前面有系数 1a\frac{1}{a}a1​，哪些没有！

  1. 反正切推广（有系数）：

    ∫dxx2+a2=1aarctan⁡xa+C\int \frac{dx}{x^2 + a^2} = \frac{1}{a} \arctan \frac{x}{a} + C∫x2+a2dx​=a1​arctanax​+C

    (易错点：千万别忘了前面的 1a\frac{1}{a}a1​)

  2. 反正弦推广（无系数）：

    ∫dxa2−x2=arcsin⁡xa+C\int \frac{dx}{\sqrt{a^2 - x^2}} = \arcsin \frac{x}{a} + C∫a2−x2​dx​=arcsinax​+C

    (记忆：arcsin⁡\arcsinarcsin 也就是“罪”，罪人是不需要穿这一件 1a\frac{1}{a}a1​ 的衣服的)

3. “长对数”与有理函数积分

这两类积分形式复杂，但经常出现在分母根式的积分题中。

  1. “长对数”公式：

    ∫dxx2±a2=ln⁡∣x+x2±a2∣+C\int \frac{dx}{\sqrt{x^2 \pm a^2}} = \ln|x + \sqrt{x^2 \pm a^2}| + C∫x2±a2​dx​=ln∣x+x2±a2​∣+C

    (无论根号里是加还是减，对数里面都是 xxx 加上那个根式)

  2. 平方差分式（有理函数）：

    ∫1a2−x2 dx=12aln⁡∣a+xa−x∣+C\int \frac{1}{a^2 - x^2} \, dx = \frac{1}{2a} \ln \left| \frac{a+x}{a-x} \right| + C∫a2−x21​dx=2a1​ln​a−xa+x​​+C

    (记忆：这是部分分式分解的结果 1a2−x2=12a(1a−x+1a+x)\frac{1}{a^2-x^2} = \frac{1}{2a}(\frac{1}{a-x} + \frac{1}{a+x})a2−x21​=2a1​(a−x1​+a+x1​))

  • 生成 arctan⁡\arctanarctan 和 ln⁡\lnln (有理分式) 的： 前面有 1a\frac{1}{a}a1​ 或 12a\frac{1}{2a}2a1​。
