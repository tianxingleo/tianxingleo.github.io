不定积分的“拆项（加一个减一个）”技巧

1. 核心口诀

“抄分母，加一个减一个”

  • 含义：观察分母中的项（例如 x2+1x^2+1x2+1 和 x2x^2x2），在分子上构造出这些项的差，从而把 111 变成 (1+x2)−x2(1+x^2) - x^2(1+x2)−x2。

2. 适用场景

  • 当被积函数是有理分式 (1P(x)Q(x)\frac{1}{P(x)Q(x)}P(x)Q(x)1​)。
  • 分子是常数（通常是 1）。
  • 分母中的两个因式只相差一个单项式（例如 x2x^2x2 和 1+x21+x^21+x2 相差 x2x^2x2；或者 xxx 和 1+x1+x1+x）。

3. 解题逻辑推导

这道题运用了两次拆分技巧，像“剥洋葱”一样把复杂分式化简。

第一步：初次拆分

  • 目标：消去分母中复杂的 (x2+1)(x^2+1)(x2+1) 或 x4x^4x4。

  • 操作：将分子 111 变为 (1+x2)−x2(1+x^2) - x^2(1+x2)−x2。

    1x4(x2+1)=(1+x2)−x2x4(x2+1)=1+x2x4(1+x2)−x2x4(1+x2) \frac{1}{x^4(x^2+1)} = \frac{(1+x^2) - x^2}{x^4(x^2+1)} = \frac{1+x^2}{x^4(1+x^2)} - \frac{x^2}{x^4(1+x^2)}x4(x2+1)1​=x4(x2+1)(1+x2)−x2​=x4(1+x2)1+x2​−x4(1+x2)x2​

  • 结果：1x4−1x2(1+x2)\frac{1}{x^4} - \frac{1}{x^2(1+x^2)}x41​−x2(1+x2)1​

第二步：再次拆分（递归使用）

  • 对于剩下的难点 1x2(1+x2)\frac{1}{x^2(1+x^2)}x2(1+x2)1​，再次使用同一种方法。

  • 操作：将分子 111 再次变为 (1+x2)−x2(1+x^2) - x^2(1+x2)−x2。

    1x2(1+x2)=(1+x2)−x2x2(1+x2)=1x2−11+x2 \frac{1}{x^2(1+x^2)} = \frac{(1+x^2) - x^2}{x^2(1+x^2)} = \frac{1}{x^2} - \frac{1}{1+x^2}x2(1+x2)1​=x2(1+x2)(1+x2)−x2​=x21​−1+x21​

第三步：合并积分项

  • 注意符号的变化（减去整个第二步的结果）：

    原式=∫1x4dx−(∫1x2dx−∫11+x2dx) \text{原式} = \int \frac{1}{x^4} dx - \left( \int \frac{1}{x^2} dx - \int \frac{1}{1+x^2} dx \right)原式=∫x41​dx−(∫x21​dx−∫1+x21​dx) =∫x−4dx−∫x−2dx+∫11+x2dx = \int x^{-4} dx - \int x^{-2} dx + \int \frac{1}{1+x^2} dx=∫x−4dx−∫x−2dx+∫1+x21​dx

4. 技巧优势

  • 速度快：不需要列方程组求解 A, B, C, D。
  • 直观：利用代数恒等变形，直接利用基本积分公式。

∫tan⁡2xdx\int \tan^2 x dx∫tan2xdx 的解法

1. 核心难点

  • 在基本积分表（公式表）中，没有直接针对 tan⁡2x\tan^2 xtan2x 的积分公式。
  • 破局点：我们需要把它变形为基本积分表中存在的形式。我们知道 (tan⁡x)′=sec⁡2x(\tan x)' = \sec^2 x(tanx)′=sec2x，所以 sec⁡2x\sec^2 xsec2x 是可以直接积分的。

2. 关键公式推导（笔记右上角红色部分）

如果不记得直接的变换公式，可以从最基础的勾股定理推导：

  1. 基础公式：sin⁡2x+cos⁡2x=1\sin^2 x + \cos^2 x = 1sin2x+cos2x=1

  2. 变形操作：等式两边同时除以 cos⁡2x\cos^2 xcos2x

  3. 得到恒等式：

    tan⁡2x+1=sec⁡2x\tan^2 x + 1 = \sec^2 xtan2x+1=sec2x

    即：

    tan⁡2x=sec⁡2x−1\tan^2 x = \sec^2 x - 1tan2x=sec2x−1

3. 解题步骤

利用上述恒等式，将被积函数进行替换：

I=∫tan⁡2x dx=∫(sec⁡2x−1) dx（代换）=∫sec⁡2x dx−∫1 dx（拆分）=tan⁡x−x+C\begin{aligned} I &= \int \tan^2 x \, dx \\ &= \int (\sec^2 x - 1) \, dx \quad \text{（代换）}\\ &= \int \sec^2 x \, dx - \int 1 \, dx \quad \text{（拆分）}\\ &= \tan x - x + C \end{aligned}I​=∫tan2xdx=∫(sec2x−1)dx（代换）=∫sec2xdx−∫1dx（拆分）=tanx−x+C​

💡 技巧总结：三角函数的“降维打击”

  • 适用场景：当你遇到 tan⁡2x\tan^2 xtan2x 或者 cot⁡2x\cot^2 xcot2x 的积分时，永远优先考虑**“加1减1”**的恒等变形。
    • 对于 tan⁡2x\tan^2 xtan2x，用 sec⁡2x−1\sec^2 x - 1sec2x−1。
    • 对于 cot⁡2x\cot^2 xcot2x，用 csc⁡2x−1\csc^2 x - 1csc2x−1。

积分中的“拆项抵消法”

这是一道利用 分部积分 进行 互相抵消 的经典题目。

1. 题目全貌

∫xex(x+1)2dx\int \frac{xe^x}{(x+1)^2} dx∫(x+1)2xex​dx

2. 解题三部曲

第一步：凑项拆分（分子变形）

  • 观察：分母是 (x+1)2(x+1)^2(x+1)2，分子只有 xxx。为了能约分，我们在分子上“凑”出 (x+1)(x+1)(x+1)。

  • 操作：将 xxx 写成 (x+1)−1(x+1) - 1(x+1)−1。

    原式=∫(x+1)−1(x+1)2exdx\text{原式} = \int \frac{(x+1)-1}{(x+1)^2} e^x dx原式=∫(x+1)2(x+1)−1​exdx

  • 结果：拆成两项积分。

    =∫exx+1dx⏟积分A−∫ex(x+1)2dx⏟积分B= \underbrace{\int \frac{e^x}{x+1} dx}_{\text{积分A}} - \underbrace{\int \frac{e^x}{(x+1)^2} dx}_{\text{积分B}}=积分A∫x+1ex​dx​​−积分B∫(x+1)2ex​dx​​

此时的心态：这两个积分如果单独算，都算不出来（因为它们属于积不出初等函数的类型）。但别急，它们之间肯定有关系。

第二步：凑微分（重点！）

  • 观察：积分 B 中的 1(x+1)2\frac{1}{(x+1)^2}(x+1)21​ 其实是 −1x+1-\frac{1}{x+1}−x+11​ 的导数。

  • 公式：

    −1u2du=d(1u)-\frac{1}{u^2} du = d\left(\frac{1}{u}\right)−u21​du=d(u1​)

  • 变形：将积分 B 中的 −1(x+1)2dx-\frac{1}{(x+1)^2} dx−(x+1)21​dx 变成 d(1x+1)d(\frac{1}{x+1})d(x+11​)。

    −∫ex(x+1)2dx=+∫exd(1x+1)-\int \frac{e^x}{(x+1)^2} dx = +\int e^x d\left(\frac{1}{x+1}\right)−∫(x+1)2ex​dx=+∫exd(x+11​)

    (注意：这一步把减号吸收到微分里去了，变成了加号)

第三步：分部积分与抵消

  • 对变形后的积分 B 使用分部积分法 ∫udv=uv−∫vdu\int u dv = uv - \int v du∫udv=uv−∫vdu：

    • 令 u=exu = e^xu=ex， dv=d(1x+1)dv = d(\frac{1}{x+1})dv=d(x+11​)

    • 则 v=1x+1v = \frac{1}{x+1}v=x+11​

    • 展开得到：

      ∫exd(1x+1)=ex⋅1x+1−∫1x+1d(ex)\int e^x d\left(\frac{1}{x+1}\right) = e^x \cdot \frac{1}{x+1} - \int \frac{1}{x+1} d(e^x)∫exd(x+11​)=ex⋅x+11​−∫x+11​d(ex)

    • 因为 d(ex)=exdxd(e^x) = e^x dxd(ex)=exdx，所以这一项变成了：

      −∫exx+1dx- \int \frac{e^x}{x+1} dx−∫x+1ex​dx

  • 见证奇迹的时刻：将结果代回原式。

    原式=∫exx+1dx⏟积分A+(exx+1−∫exx+1dx⏟积分A的克星)\text{原式} = \underbrace{\int \frac{e^x}{x+1} dx}_{\text{积分A}} + \left( \frac{e^x}{x+1} - \underbrace{\int \frac{e^x}{x+1} dx}_{\text{积分A的克星}} \right)原式=积分A∫x+1ex​dx​​+​x+1ex​−积分A的克星∫x+1ex​dx​​​

  • 最终结果：首尾两项积分一正一负，互相抵消，只剩下中间部分。

    =exx+1+C= \frac{e^x}{x+1} + C=x+1ex​+C

💡 技巧总结：抵消模型

这种题型被称为 “无中生有，自相残杀”，其背后的数学原理是一个重要结论：

∫ex[f(x)+f′(x)]dx=exf(x)+C\int e^x [f(x) + f'(x)] dx = e^x f(x) + C∫ex[f(x)+f′(x)]dx=exf(x)+C
  • 在本题中：
    • f(x)=1x+1f(x) = \frac{1}{x+1}f(x)=x+11​
    • f′(x)=−1(x+1)2f'(x) = -\frac{1}{(x+1)^2}f′(x)=−(x+1)21​
  • 题目一开始给出的 x(x+1)2\frac{x}{(x+1)^2}(x+1)2x​ 其实就是 1x+1−1(x+1)2\frac{1}{x+1} - \frac{1}{(x+1)^2}x+11​−(x+1)21​，也就是 f(x)+f′(x)f(x) + f'(x)f(x)+f′(x) 的形式。
  • 做题策略：遇到 exe^xex 乘以复杂分式，试着把分式拆开，对其中一项分部积分，看看能不能抵消掉另一项。
