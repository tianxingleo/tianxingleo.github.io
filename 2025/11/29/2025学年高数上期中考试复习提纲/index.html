第一章：极限与连续

三角函数

1. 倒数与商数关系（基础铺垫）

这些在化简被积函数时最常用，比如把 tan⁡\tantan 拆成 sin⁡/cos⁡\sin/\cossin/cos。

  • tan⁡x=sin⁡xcos⁡x\tan x = \frac{\sin x}{\cos x}tanx=cosxsinx​
  • cot⁡x=cos⁡xsin⁡x=1tan⁡x\cot x = \frac{\cos x}{\sin x} = \frac{1}{\tan x}cotx=sinxcosx​=tanx1​
  • sec⁡x=1cos⁡x\sec x = \frac{1}{\cos x}secx=cosx1​
  • csc⁡x=1sin⁡x\csc x = \frac{1}{\sin x}cscx=sinx1​

2. 平方关系（勾股性质）

解不定积分时，常用于凑微分或三角代换（如 x=atan⁡tx=a\tan tx=atant）。

  1. sin⁡2x+cos⁡2x=1\sin^2 x + \cos^2 x = 1sin2x+cos2x=1 （万恶之源，最常用）
  2. 1+tan⁡2x=sec⁡2x1 + \tan^2 x = \sec^2 x1+tan2x=sec2x （重点：求 ∫sec⁡2xdx\int \sec^2 x dx∫sec2xdx 或 ∫tan⁡2xdx\int \tan^2 x dx∫tan2xdx 时必用）
  3. 1+cot⁡2x=csc⁡2x1 + \cot^2 x = \csc^2 x1+cot2x=csc2x

3. 倍角公式（简化计算的核心）

在求极限和化简导数时非常有用。

  • 正弦倍角：

    sin⁡2x=2sin⁡xcos⁡x\sin 2x = 2\sin x \cos xsin2x=2sinxcosx

  • 余弦倍角（有三种形式，灵活选择）：

    cos⁡2x=cos⁡2x−sin⁡2x\cos 2x = \cos^2 x - \sin^2 xcos2x=cos2x−sin2x cos⁡2x=2cos⁡2x−1\cos 2x = 2\cos^2 x - 1cos2x=2cos2x−1 cos⁡2x=1−2sin⁡2x\cos 2x = 1 - 2\sin^2 xcos2x=1−2sin2x

  • 正切倍角：

    tan⁡2x=2tan⁡x1−tan⁡2x\tan 2x = \frac{2\tan x}{1 - \tan^2 x}tan2x=1−tan2x2tanx​

4. 降幂公式（积分的救命稻草） ★★★

这是高数中最重要的一组变形！遇到 sin⁡2x\sin^2 xsin2x 或 cos⁡2x\cos^2 xcos2x 的积分，必须降幂才能积出来。

  • 正弦降幂：

    sin⁡2x=1−cos⁡2x2\sin^2 x = \frac{1 - \cos 2x}{2}sin2x=21−cos2x​

  • 余弦降幂：

    cos⁡2x=1+cos⁡2x2\cos^2 x = \frac{1 + \cos 2x}{2}cos2x=21+cos2x​

记忆口诀： “正弦减，余弦加”（看分子中间的符号）。

5. 和差角公式（展开项）

用于证明导数公式或处理复合角的极限。

  • sin⁡(α±β)=sin⁡αcos⁡β±cos⁡αsin⁡β\sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\betasin(α±β)=sinαcosβ±cosαsinβ
  • cos⁡(α±β)=cos⁡αcos⁡β∓sin⁡αsin⁡β\cos(\alpha \pm \beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\betacos(α±β)=cosαcosβ∓sinαsinβ （**注意符号相反**）
  • tan⁡(α±β)=tan⁡α±tan⁡β1∓tan⁡αtan⁡β\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha\tan\beta}tan(α±β)=1∓tanαtanβtanα±tanβ​

当你遇到 ∫sin⁡3xcos⁡5x dx\int \sin 3x \cos 5x \, dx∫sin3xcos5xdx 这种“不同频率三角函数相乘”的积分时，必须用积化和差把它们拆开。

间断点

核心分类逻辑图

设 x0x_0x0​ 是函数 f(x)f(x)f(x) 的间断点（即函数在该点没定义，或者极限值不等于函数值）。

  1. 第一步：算出左极限 f(x0−)f(x_0^-)f(x0−​) 和 右极限 f(x0+)f(x_0^+)f(x0+​)。
  2. 第二步：判断它们是否都存在（即是否都是有限常数）。
    • 都存在 ⇒\Rightarrow⇒ 第一类间断点 (性质比较温和，图像只是断开)。
    • 至少有一个不存在 ⇒\Rightarrow⇒ 第二类间断点 (性质比较恶劣，图像通常跑向无穷或剧烈震荡)。

一、 第一类间断点 (First Kind Discontinuity)

定义： 左右极限都存在。

这意味着函数在这一点的“断裂”是可以修补或者是有限的跳跃。

它细分为两种：

1. 可去间断点 (Removable Discontinuity)
  • 特征： 左极限 === 右极限 (≠\neq= 函数值，或者函数在此无定义)。
  • 直观理解： 图像上有一个“空心洞”。如果我们把这个洞补上（重新定义这一点的函数值），它就变成连续函数了。
  • 经典例子： f(x)=sin⁡xxf(x) = \frac{\sin x}{x}f(x)=xsinx​ 在 x=0x=0x=0 处。
    • lim⁡x→0−sin⁡xx=1\lim_{x\to 0^-} \frac{\sin x}{x} = 1limx→0−​xsinx​=1
    • lim⁡x→0+sin⁡xx=1\lim_{x\to 0^+} \frac{\sin x}{x} = 1limx→0+​xsinx​=1
    • 左右极限相等，所以是可去间断点。
2. 跳跃间断点 (Jump Discontinuity)
  • 特征： 左极限 ≠\neq= 右极限。
  • 直观理解： 图像在这里断开了，像台阶一样“跳”了一下。无法通过补一个点让它连续。
  • 经典例子： f(x)=arctan⁡(1x)f(x) = \arctan(\frac{1}{x})f(x)=arctan(x1​) 在 x=0x=0x=0 处。
    • lim⁡x→0−arctan⁡(1x)=−π2\lim_{x\to 0^-} \arctan(\frac{1}{x}) = -\frac{\pi}{2}limx→0−​arctan(x1​)=−2π​
    • lim⁡x→0+arctan⁡(1x)=π2\lim_{x\to 0^+} \arctan(\frac{1}{x}) = \frac{\pi}{2}limx→0+​arctan(x1​)=2π​
    • 左右极限都在，但不相等，所以是跳跃间断点。

二、 第二类间断点 (Second Kind Discontinuity)

定义： 左右极限中，至少有一个不存在（包括等于 ∞\infty∞）。

这意味着函数在这一点“失控”了。

它常见的有以下几种：

1. 无穷间断点 (Infinite Discontinuity)
  • 特征： 极限为 ∞\infty∞（无穷大）。
  • 直观理解： 图像贴着垂直渐近线向上冲或向下冲。
  • 经典例子： f(x)=tan⁡xf(x) = \tan xf(x)=tanx 在 x=π2x=\frac{\pi}{2}x=2π​ 处，或者 f(x)=1xf(x)=\frac{1}{x}f(x)=x1​ 在 x=0x=0x=0 处。
    • lim⁡x→0+1x=+∞\lim_{x\to 0^+} \frac{1}{x} = +\inftylimx→0+​x1​=+∞（极限不存在）。
    • 只要有一个是无穷大，就是第二类。
2. 振荡间断点 (Oscillating Discontinuity)
  • 特征： 当 x→x0x \to x_0x→x0​ 时，函数值在两个数之间无限快地来回摆动，没有趋向于任何一个固定的数。
  • 直观理解： 越靠近这一点，波浪越密，根本停不下来。
  • 经典例子： f(x)=sin⁡(1x)f(x) = \sin(\frac{1}{x})f(x)=sin(x1​) 在 x=0x=0x=0 处。
    • 当 x→0x \to 0x→0 时，1x→∞\frac{1}{x} \to \inftyx1​→∞，正弦函数在 -1 到 1 之间疯狂震荡，极限不存在。

求解递推数列极限

方法一：单调有界准则 (Monotone Bounded Convergence Theorem)

(对应图片 2025-11-28 173037.png)

这是处理数列极限最通用的标准方法。

核心逻辑：

若数列 单调（增或减） 且 有界（有上界或下界），则极限必存在。

解题步骤：

  1. 证有界性 (Boundedness)：
    • 常用方法：不等式缩放、数学归纳法。
    • 笔记案例： 通过观察通项 an+1=3⋅1+an3+ana_{n+1} = 3 \cdot \frac{1+a_n}{3+a_n}an+1​=3⋅3+an​1+an​​，发现分式部分 <1<1<1，从而得出 an+1<3a_{n+1} < 3an+1​<3，证明了有界性。
  2. 证单调性 (Monotonicity)：
    • 常用方法：做差法 (xn+1−xnx_{n+1} - x_nxn+1​−xn​)、做商法 (xn+1/xnx_{n+1}/x_nxn+1​/xn​)。
    • 高阶技巧（笔记重点）： 利用导数判别。
      • 构造函数 f(x)f(x)f(x) 使得 xn+1=f(xn)x_{n+1} = f(x_n)xn+1​=f(xn​)。
      • 若 f′(x)>0f'(x) > 0f′(x)>0，则数列 xn{x_n}xn​ 必定单调（具体是递增还是递减取决于前两项 x2x_2x2​ 与 x1x_1x1​ 的大小关系）。
  3. 求极限值：
    • 设 lim⁡n→∞an=A\lim_{n \to \infty} a_n = Alimn→∞​an​=A。
    • 对递推式两边取极限，解方程 A=f(A)A = f(A)A=f(A)。
    • 笔记案例结果： A=3A = \sqrt{3}A=3​。

方法二：压缩映射原理 (Contraction Mapping Principle)

(对应图片 2025-11-28 173047.png)

当数列不单调（例如摆动趋近）或者单调性难以证明时，使用此方法。通常适用于 ∣f′(x)∣<1|f'(x)| < 1∣f′(x)∣<1 的情况。

      ∣xn+1−a∣=∣f(xn)−f(a)∣=∣f′(ξ)∣⋅∣xn−a∣≤k∣xn−a∣|x_{n+1} - a| = |f(x_n) - f(a)| = |f'(\xi)| \cdot |x_n - a| \le k |x_n - a|∣xn+1​−a∣=∣f(xn​)−f(a)∣=∣f′(ξ)∣⋅∣xn​−a∣≤k∣xn​−a∣

    • 因为 k<1k < 1k<1，当 n→∞n \to \inftyn→∞ 时，kn→0k^n \to 0kn→0，故 ∣xn+1−a∣→0|x_{n+1} - a| \to 0∣xn+1​−a∣→0，即 lim⁡xn=a\lim x_n = alimxn​=a。

导数定义

❌ 错误命题 (False Statement)

命题： 若极限 lim⁡h→0f(2h)−f(h)h\lim_{h \to 0} \frac{f(2h) - f(h)}{h}limh→0​hf(2h)−f(h)​ 存在，则 f(x)f(x)f(x) 在 x=0x=0x=0 处可导。

结论： ❌ 错误。

🔬 核心分析

导数在 x=0x=0x=0 处的标准定义（若存在）为：

f′(0)=lim⁡h→0f(0+h)−f(0)hf'(0) = \lim_{h \to 0} \frac{f(0+h) - f(0)}{h}f′(0)=h→0lim​hf(0+h)−f(0)​

只有当上述标准极限存在时，才称函数 f(x)f(x)f(x) 在 x=0x=0x=0 处可导。

错误原因：

题中的极限 lim⁡h→0f(2h)−f(h)h\lim_{h \to 0} \frac{f(2h) - f(h)}{h}limh→0​hf(2h)−f(h)​ 仅要求 f(x)f(x)f(x) 在 x=0x=0x=0 附近（即 x≠0x \neq 0x=0）的值有良好的行为，但它没有强制性要求 f(x)f(x)f(x) 在 x=0x=0x=0 处必须连续，而可导的前提是必须连续。

第一步：验证题中极限存在
