错题本：高等数学 - 极限与变限积分求导

1. 题目序号：3

【原题描述】
设 lim⁡x→0xf(x)+ln⁡(1+x)x2=12\lim_{x\to 0} \frac{xf(x) + \ln(1+x)}{x^2} = \frac{1}{2}limx→0​x2xf(x)+ln(1+x)​=21​，则 lim⁡x→0f(x)+1x=()\lim_{x\to 0} \frac{f(x)+1}{x} = (\quad)limx→0​xf(x)+1​=()。
选项：(A) 32\frac{3}{2}23​ (B) 0 © 12\frac{1}{2}21​ (D) 1

【手写解题步骤】

  1. 泰勒展开：将 ln⁡(1+x)\ln(1+x)ln(1+x) 进行泰勒展开：ln⁡(1+x)=x−12x2+o(x2) \ln(1+x) = x - \frac{1}{2}x^2 + o(x^2) ln(1+x)=x−21​x2+o(x2)
  2. 代入极限式：将展开式代入原已知极限条件：12=lim⁡x→0xf(x)+x−12x2+o(x2)x2 \frac{1}{2} = \lim_{x\to 0} \frac{xf(x) + x - \frac{1}{2}x^2 + o(x^2)}{x^2} 21​=x→0lim​x2xf(x)+x−21​x2+o(x2)​
  3. 整理分子：12=lim⁡x→0x(f(x)+1)−12x2+o(x2)x2 \frac{1}{2} = \lim_{x\to 0} \frac{x(f(x)+1) - \frac{1}{2}x^2 + o(x^2)}{x^2} 21​=x→0lim​x2x(f(x)+1)−21​x2+o(x2)​
  4. 拆分极限：12=lim⁡x→0f(x)+1x−12 \frac{1}{2} = \lim_{x\to 0} \frac{f(x)+1}{x} - \frac{1}{2} 21​=x→0lim​xf(x)+1​−21​
  5. 求解：lim⁡x→0f(x)+1x=12+12=1 \lim_{x\to 0} \frac{f(x)+1}{x} = \frac{1}{2} + \frac{1}{2} = 1 x→0lim​xf(x)+1​=21​+21​=1 【答案】 (D)

【解题思路特点与考点】

  • 考点：泰勒公式在计算极限中的应用、极限的逆问题。
  • 特点：当题目中出现 ln⁡(1+x)\ln(1+x)ln(1+x) 与 x2x^2x2 分母同阶或更高阶时，直接使用等价无穷小（xxx）可能会丢失高阶项（−12x2-\frac{1}{2}x^2−21​x2），导致精度不够。必须展开到与分母同阶的项数。此题关键在于准确写出 ln⁡(1+x)\ln(1+x)ln(1+x) 的二阶展开式。

2. 题目序号：7

【原题描述】

lim⁡x→0tan⁡(tan⁡x)−xx3=()\lim_{x\to 0} \frac{\tan(\tan x) - x}{x^3} = (\quad)limx→0​x3tan(tanx)−x​=()。

选项：(A) −23-\frac{2}{3}−32​ (B) 13\frac{1}{3}31​ © −13-\frac{1}{3}−31​ (D) 23\frac{2}{3}32​

【手写解题步骤】

  1. 添项拆分：在分子中加减 tan⁡x\tan xtanx，构造已知极限形式：原式=lim⁡x→0(tan⁡(tan⁡x)−tan⁡xx3+tan⁡x−xx3) 原式 = \lim_{x\to 0} \left( \frac{\tan(\tan x) - \tan x}{x^3} + \frac{\tan x - x}{x^3} \right) 原式=x→0lim​(x3tan(tanx)−tanx​+x3tanx−x​) (注：图片中标注了 x3∼tan⁡3xx^3 \sim \tan^3 xx3∼tan3x)
  2. 变量代换（第一部分）：
    对于第一项，令 tan⁡x=t\tan x = ttanx=t，当 x→0x\to 0x→0 时 t→0t\to 0t→0，且分母 x3∼t3x^3 \sim t^3x3∼t3：lim⁡x→0tan⁡(tan⁡x)−tan⁡xx3=lim⁡t→0tan⁡t−tt3 \lim_{x\to 0} \frac{\tan(\tan x) - \tan x}{x^3} = \lim_{t\to 0} \frac{\tan t - t}{t^3} x→0lim​x3tan(tanx)−tanx​=t→0lim​t3tant−t​
  3. 合并计算：
    因为两部分极限形式相同，故原式变为：=2lim⁡x→0tan⁡x−xx3 = 2 \lim_{x\to 0} \frac{\tan x - x}{x^3} =2x→0lim​x3tanx−x​
  4. 利用常用结论：
    已知泰勒展开结论 tan⁡x−x∼13x3\tan x - x \sim \frac{1}{3}x^3tanx−x∼31​x3，故：=2×13=23 = 2 \times \frac{1}{3} = \frac{2}{3} =2×31​=32​ 【答案】 (D)

【解题思路特点与考点】

  • 考点：复合函数的极限、凑项法（加一项减一项）、等价无穷小代换。
  • 特点：直接使用泰勒展开较繁琐。通过“减去中间变量 tan⁡x\tan xtanx 再加上 tan⁡x\tan xtanx”的技巧，将复合函数 tan⁡(tan⁡x)\tan(\tan x)tan(tanx) 转化为两个结构相同的基本极限 tan⁡u−uu3\frac{\tan u - u}{u^3}u3tanu−u​ 的和。利用代换思想简化计算。

3. 题目序号：8

【原题描述】
极限 lim⁡x→0(1x2−1tan⁡2x)\lim_{x\to 0} (\frac{1}{x^2} - \frac{1}{\tan^2 x})limx→0​(x21​−tan2x1​) 的值为 ()(\quad)()。
选项：(A) 13\frac{1}{3}31​ (B) 23\frac{2}{3}32​ © −13-\frac{1}{3}−31​ (D) −23-\frac{2}{3}−32​

【手写解题步骤】

  1. 通分：原式=lim⁡x→0tan⁡2x−x2x2tan⁡2x 原式 = \lim_{x\to 0} \frac{\tan^2 x - x^2}{x^2 \tan^2 x} 原式=x→0lim​x2tan2xtan2x−x2​
  2. 分母等价代换：
    因 tan⁡x∼x\tan x \sim xtanx∼x，故分母 x2tan⁡2x∼x4x^2 \tan^2 x \sim x^4x2tan2x∼x4：=lim⁡x→0tan⁡2x−x2x4 = \lim_{x\to 0} \frac{\tan^2 x - x^2}{x^4} =x→0lim​x4tan2x−x2​
  3. 分子因式分解：=lim⁡x→0(tan⁡x+x)(tan⁡x−x)x4 = \lim_{x\to 0} \frac{(\tan x + x)(\tan x - x)}{x^4} =x→0lim​x4(tanx+x)(tanx−x)​
  4. 拆分极限：=lim⁡x→0tan⁡x+xx⋅lim⁡x→0tan⁡x−xx3 = \lim_{x\to 0} \frac{\tan x + x}{x} \cdot \lim_{x\to 0} \frac{\tan x - x}{x^3} =x→0lim​xtanx+x​⋅x→0lim​x3tanx−x​
  5. 计算各部分：
    • 前部分：lim⁡x→0(tan⁡xx+1)=1+1=2\lim_{x\to 0} (\frac{\tan x}{x} + 1) = 1 + 1 = 2limx→0​(xtanx​+1)=1+1=2
    • 后部分（常用结论）：13\frac{1}{3}31​
    • 结果：2×13=232 \times \frac{1}{3} = \frac{2}{3}2×31​=32​
      【答案】 (D) （注：虽然图片勾选不明显，但计算结果为 2/3，对应 D 或 B 取决于排版，根据计算过程应选 2/3）

【解题思路特点与考点】

  • 考点：∞−∞\infty - \infty∞−∞ 型极限的计算、通分、平方差公式、等价无穷小。
  • 特点：遇到分式差的极限，首选通分。通分后利用代数公式（平方差）将高阶无穷小拆分为低阶因子的乘积，再分别求极限。注意分母代换时要精准（x2⋅x2=x4x^2 \cdot x^2 = x^4x2⋅x2=x4）。

4. 题目序号：12

【原题描述】

lim⁡x→0((cos⁡x)1x2+tan⁡x−xx3)=()\lim_{x\to 0} \left( (\cos x)^{\frac{1}{x^2}} + \frac{\tan x - x}{x^3} \right) = (\quad)limx→0​((cosx)x21​+x3tanx−x​)=()。

选项：(A) e−12+13e^{-\frac{1}{2}} + \frac{1}{3}e−21​+31​ (B) e−12−13e^{-\frac{1}{2}} - \frac{1}{3}e−21​−31​ © e12+13e^{\frac{1}{2}} + \frac{1}{3}e21​+31​ (D) e12−13e^{\frac{1}{2}} - \frac{1}{3}e21​−31​

【手写解题步骤】

  1. 拆分两项：
    第二项已知 lim⁡x→0tan⁡x−xx3=13\lim_{x\to 0} \frac{\tan x - x}{x^3} = \frac{1}{3}limx→0​x3tanx−x​=31​。重点计算第一项。
  2. 幂指函数极限：
    令 L1=lim⁡x→0(cos⁡x)1x2L_1 = \lim_{x\to 0} (\cos x)^{\frac{1}{x^2}}L1​=limx→0​(cosx)x21​。
    利用公式 uv=evln⁡uu^v = e^{v \ln u}uv=evlnu：L1=elim⁡x→01x2ln⁡(cos⁡x) L_1 = e^{\lim_{x\to 0} \frac{1}{x^2} \ln(\cos x)} L1​=elimx→0​x21​ln(cosx)
  3. 等价无穷小代换：
    当 x→0x\to 0x→0 时，ln⁡(cos⁡x)∼cos⁡x−1∼−12x2\ln(\cos x) \sim \cos x - 1 \sim -\frac{1}{2}x^2ln(cosx)∼cosx−1∼−21​x2。L1=elim⁡x→01x2(−12x2)=e−12 L_1 = e^{\lim_{x\to 0} \frac{1}{x^2} (-\frac{1}{2}x^2)} = e^{-\frac{1}{2}} L1​=elimx→0​x21​(−21​x2)=e−21​
  4. 合并结果：原式=e−12+13 原式 = e^{-\frac{1}{2}} + \frac{1}{3} 原式=e−21​+31​ 【答案】 (A)

【解题思路特点与考点】

  • 考点：幂指函数极限（1∞1^\infty1∞ 型）、常用等价无穷小 ln⁡(cos⁡x)∼−12x2\ln(\cos x) \sim -\frac{1}{2}x^2ln(cosx)∼−21​x2。
  • 特点：题目由两部分组成，考察了极限的四则运算法则。幂指函数部分采用取对数改写成指数形式的方法最为稳妥。需熟记 cos⁡x−1\cos x - 1cosx−1 的等价无穷小。

5. 知识点总结：变上限积分求导（图片 6 & 7）

一、复合函数变上限求导

  • 核心公式：
    若 F(x)=∫aφ(x)f(t)dtF(x) = \int_a^{\varphi(x)} f(t) dtF(x)=∫aφ(x)​f(t)dt，则：F′(x)=f(φ(x))⋅φ′(x) F'(x) = f(\varphi(x)) \cdot \varphi'(x) F′(x)=f(φ(x))⋅φ′(x)
  • 口诀：“代入上限，乘以（上限的）导数”。
  • 例题：∫0sin⁡xf(t)dt\int_0^{\sin x} f(t) dt∫0sinx​f(t)dt
    • 步骤：
      1. 代入：把积分里的 ttt 全部换成上限 sin⁡x\sin xsinx。
      2. 求导：对上限 sin⁡x\sin xsinx 求导得 cos⁡x\cos xcosx。
      3. 相乘：结果为 f(sin⁡x)⋅cos⁡xf(\sin x) \cdot \cos xf(sinx)⋅cosx。

二、被积函数含参变量（换元法）

  • 问题类型：求导 ∫0xf(x−t)dt\int_0^x f(x-t) dt∫0x​f(x−t)dt。被积函数内部含有 xxx，不能直接求导。
  • 处理方法（换元法）：
    目的是把 xxx 从函数内部“移”到积分限或外部。
    1. 令 u=x−tu = x - tu=x−t，则 t=x−ut = x - ut=x−u，且 dt=−dudt = -dudt=−du。
    2. 换限：当 t=0t=0t=0 时 u=xu=xu=x；当 t=xt=xt=x 时 u=0u=0u=0。
    3. 变换积分：∫0xf(x−t)dt=∫x0f(u)(−du)=∫0xf(u)du \int_0^x f(x-t) dt = \int_x^0 f(u)(-du) = \int_0^x f(u) du ∫0x​f(x−t)dt=∫x0​f(u)(−du)=∫0x​f(u)du
    4. 求导：
      变为标准形式后直接求导：ddx∫0xf(u)du=f(x) \frac{d}{dx} \int_0^x f(u) du = f(x) dxd​∫0x​f(u)du=f(x)
  • 考点：积分变限函数的求导规则、换元积分法在变限积分中的应用。这是处理卷积型变限积分的标准步骤。
